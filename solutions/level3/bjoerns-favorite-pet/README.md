## Bjoern's Favorite Pet
### Challenge: Reset the password of Bjoern's OWASP account via the Forgot Password mechanism with the original answer to his security question.

1. Visit http://\<vm_ip\>:3000/#/forgot-password and provide bjoern@owasp.org as your Email.
2. You will notice that the security question Bjoern chose is Name of your favorite pet?
3. Go to the photo wall and search for a photo uploaded by Twitter user bkimminich
4. Visit this user's Twitter profile
5. Scroll down until you find a post that names his cat from September 25, 2021
6. Visit http://\<vm_ip\>:3000/#/forgot-password and once more provide bjoern@owasp.org as your Email.
7. Provide the name of the cat you found in the Twitter post (Zaya) as the answer to the security question
8. Then type any New Password and matching Repeat New Password
9. Click Change to solve this challenge
