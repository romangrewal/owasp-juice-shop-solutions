## Web3 Sandbox
### Challenge: Find an accidentally deployed code sandbox for writing smart contracts on the fly.

1. Right Click on home page > Inspect > Sources
2. Look at `main.js`
3. Search for `web3` or `sandbox` and you would observe that there is a directory path named `web3-sandbox`
4. navigate to the url `http://ipaddress:3000/#/web3-sandbox` to complete this challenge
