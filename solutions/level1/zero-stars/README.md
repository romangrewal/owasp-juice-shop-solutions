## Zero Stars 
### Challenge: Give a devastating zero-star feedback to the store.

* Navigate to Customer Feedback
* Write a comment
* DO NOT TOGGLE THE RATING SLIDER
* Solve the captcha
* Open the HTML inspector
* Find the Submit button
* Remove class `mat-mdc-button-disabled`
* Remove HTML attributes `mat-ripple-loader-disabled` and `disabled`
* Click the enabled submit button to complete this challenge
