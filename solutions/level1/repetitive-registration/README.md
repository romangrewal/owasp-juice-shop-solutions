## Repetitive Registration
### Challenge: Follow the DRY principle while registering a user.

* Navigate to registration screen, Account > Login > Not yet a customer?
* Fill out all required information except the _Password_ and _Repeat Password_ field.
* Type e.g. 12345 into the _Password_ field.
* Type same password into the _Repeat Password_ field.
* Go back to the _Password_ field and change it into any other password.
* Succesfully submit the form with the _Register_ button to solve this challenge.
