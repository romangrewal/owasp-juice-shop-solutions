## Repetitive Registration
### Challenge: Follow the DRY principle while registering a user.

1. Navigate to registration screen, Account > Login > Not yet a customer?
2. Fill out all required information except the _Password_ and _Repeat Password_ field.
3. Type e.g. 12345 into the _Password_ field.
4. Type same password into the _Repeat Password_ field.
5. Go back to the _Password_ field and change it into any other password.
6. Succesfully submit the form with the _Register_ button to solve this challenge.
