### Challenge: Inform the shop about a typosquatting imposter that dug itself deep into the frontend. (Mention the exact name of the culprit)

1. Request http://\<host\>:3000/3rdpartylicenses.txt to retrieve the 3rd party license list generated by Angular CLI by default.
2. Comb through the list to find `ngy-cookie` which is typosquatting `ngx-cookie` in npm packages.
3. Log out of the application.
4. Visit http://\<host\>:3000/#/contact
5. Submit your feedback with ngy-cookie in the comment to solve this challenge.
